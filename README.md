## 📌 Project Overview

Wine quality prediction is an important task in the **wine industry** to ensure product consistency and customer satisfaction.  
This project builds a **Machine Learning model** that predicts the quality of wine based on its **physiochemical properties**.

By analyzing chemical attributes such as acidity, sugar content, and alcohol level, the model predicts wine quality and helps improve **wine production processes and sales strategies**.

---

## 🎯 Objective

- Predict the **quality of wine** using machine learning models.
- Analyze **physiochemical properties** affecting wine quality.
- Assist producers in **improving production processes**.
- Support **data-driven decision making** in the wine industry.

---

## 📊 Dataset

The dataset used is the **Wine Quality Dataset** from the **UCI Machine Learning Repository**.

### Dataset Details
- **Total Rows:** 1599
- **Total Features:** 12
- **Wine Type:** Red & White vinho verde wines from Portugal

### Features Included

| Feature | Description |
|------|-------------|
| Fixed Acidity | Amount of non-volatile acids |
| Volatile Acidity | Amount of acetic acid |
| Citric Acid | Freshness of wine |
| Residual Sugar | Remaining sugar after fermentation |
| Chlorides | Salt content |
| Free Sulfur Dioxide | Prevents microbial growth |
| Total Sulfur Dioxide | Total SO₂ content |
| Density | Density of wine |
| pH | Acidity level |
| Sulphates | Preservative effect |
| Alcohol | Alcohol percentage |
| Quality | Target variable |
